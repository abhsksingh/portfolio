# Abhishek Kumar Singh – Portfolio

> *"Innovate. Implement. Inspire."*

A modern, glassmorphism-styled personal portfolio website showcasing my skills, projects, and professional profile.

## 🌐 Live Preview

🔗 **[https://portfolio-exns.onrender.com/](https://portfolio-exns.onrender.com/)**

## ✨ Features

- **Glassmorphism UI** – Frosted glass cards with dynamic floating orb backgrounds
- **Responsive Design** – Optimized for desktop and mobile
- **Smooth Animations** – Skill bar fills, hover effects, and floating orb animations
- **Single-File Architecture** – Fully self-contained HTML with embedded CSS (no dependencies)
- **Dark Theme** – Premium dark gradient aesthetic

## 📂 Sections

| Section      | Description                                      |
|--------------|--------------------------------------------------|
| **Hero**     | Introduction with avatar, tagline, and CTAs      |
| **About**    | Background, stats, and tech stack pill cloud      |
| **Skills**   | Core expertise with animated proficiency bars     |
| **Projects** | GitHub project cards with tags and repo links     |
| **Contact**  | Email, phone, LinkedIn, and GitHub links          |
